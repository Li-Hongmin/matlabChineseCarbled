# Matlab 中文乱码解决办法

进入matlab的目录了找到 bin目录下 lcdata.xml 这个文件


<locale name="zh_CN" encoding="GB2312" xpg_name="zh_CN.GB2312">
	<alias name="zh-CN"/>

	<alias name="zh-Hans"/>

	<alias name="zh-Hans-CN"/>

</locale>

<locale name="en_US_POSIX" encoding="GB2312" xpg_name="zh_CN.GB2312">
	<alias name="zh-CN"/>

	<alias name="zh-Hans"/>

	<alias name="zh-Hans-CN"/>

</locale>
